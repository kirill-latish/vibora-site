# Vibora Padel Tracker — Marketing Site

Static landing page, support, and privacy policy for the Vibora Padel
Tracker iOS / watchOS app.

Live at: https://kirill-latish.github.io/vibora-site/

The app source lives in a private repository; this repo only contains
the public marketing pages and is automatically deployed to GitHub
Pages on every push to `main` via `.github/workflows/deploy-pages.yml`.
